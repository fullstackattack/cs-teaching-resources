# cs-teaching-resources

CS Teaching Resources

A collection of lessons, worksheets, coding exercises, and projects designed to help high school students build strong foundations in computer science and programming.

Created and maintained by Amanda Stanley, computer science educator and software engineer.

---

## Overview

This repository is a curated library of classroom materials that focus on computational thinking, Python programming, web development, and ethical applications of technology.

Each resource is designed to be:
- **Hands-on:** Students learn by writing, debugging, and explaining real code.
- **Structured:** Lessons progress logically from fundamentals to applied projects.
- **Adaptable:** Materials can be used for lectures, self-paced learning, or online instruction.
- **Ethical:** Many activities integrate discussions about AI, data privacy, and technologyâ€™s social impact.
